# weird-stories
Weird Stories
=============

> A personal attempt to compiling a complete list of the stories part of the Cthulhu Mythos

I first started this list some years ago when I was editor at [Lovecraft Zero](https://web.archive.org/web/*/lovecraftzero.it). Back then, moved by curiosity and by the need of gatherer the tremendous literary opus produced by HPL, I began to spontaneously note down literary works referred to the Cthulhu Mythos or its elements.

### H.P. Lovecraft

* **Dagon** (1917) - `Dagon` cf. `The Shadow Over Innsmouth`
* **Nyarlathotep** (1920) - `Nyarlathotep` later turned into an Outer God
* **The Nameless City** (1921) - `Alhzred` is introduced
* **The Outsider** (?) - `Nephren-Ka` is introduced
* **The Festival** (1923) - `Kingsport`, `Daemonolatreia`, `Necronomicon`
* **The Call of Cthulhu** (1926) - `Alhazred`, `Necronomicon`
